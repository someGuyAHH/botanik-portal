# Türkiye Endemik Bitki Atlası

An interactive React application featuring a map of Turkey with detailed information about endemic plants across 81 provinces.

## Features

- **Interactive Turkey Map**: Click any of the 81 Turkish provinces to explore endemic plants
- **Region-Based Color Coding**: Provinces colored by their geographical region (7 regions total)
- **Plant Database**: Comprehensive endemic plant data with Turkish names, Latin names, and traditional uses
- **Responsive Design**: Clean, botanical-themed interface

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **turkey-map-react** - Interactive SVG map of Turkey
- **CSS** - Styling with region-based theming

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```
